# WordBank - DogeVoid Loader

## 📌 Description
This repository contains a simple loader script that automatically fetches and executes the main script from a remote source.

The script uses `loadstring` to always load the latest version directly from GitHub.

---

## 🚀 How to Use

Paste the following code into your executor:

```lua
loadstring(game:HttpGet("https://raw.githubusercontent.com/bro-pixel11/WordBank-DogeVoid/main/loader.lua"))()
